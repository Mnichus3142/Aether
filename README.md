# Aether

Aether is a ecosystem of apps, developed for making linux life easier and more productive.

## Apps

- **aetherd**: A background daemon that manages various system tasks and services.
- **aether-bar**: A customizable status bar for Linux desktops.
- **aether-launcher**: A lightweight application launcher for quick access to your favorite apps.

### Work in Progress

- **aether-osd**: An on-screen display tool for things like volume, brightness, and CapsLock.
- **aether-tray**: A tray in which you can access basic settings like wifi, bluetooth or brightness.
- **aether-lock**: An on-screen display tool enabling you to lock, hibernate or turn off your device.
- **aether-island**: A dynamic island for displaying notifications or things like music control.
